# Complition Assist

Клиентский мод для Майнкарфт 1.21.8 для показа подсказок и кастомной автозамены.

## Функции
- **Работает почти во всех полях ввода**: Текстовые поля, Чат, Наковальня, Книжка, Поиск в инвенратре, (Везде где используеться TextFieldWidget)
- **Замена**: Чтобы воспользоваться введите сивол ":" и хотя бы одну букву. Если видна хотябы одна подсказка для этой буквы то по нажатию LeftShift + Space мод заменит введенную строку на текст замены. Или нажмите ПКМ по подсказке и мод вставит замену.
- Можете включать и выключать мод через настройки

## Требования
- Minecraft 1.21.8
- Fabric Loader
- Fabric API
- Вретолет и 4 золотых нагенца
